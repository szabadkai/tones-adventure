Tone's Adventure

An Abandoned Homebrew Gameboy Game

An Experiment in Gameboy Programming

Tone's Adventure was my first and only attempt at creating a game for the Nintendo Gameboy Color. It was an actual ROM that Gameboy emulators and even real Gameboys could play. It was programmed in C, and was compiled using the GBDK compiler.

As demonstrated in the video, the player is able to explore a small cave map. The physics are fairly sophisticated. The player can run, jump, swim, and can slide down slopes.

Work on this game began in September 2011. The version you see in the video was completed in February 2012.

What happened?

Work on the project ceased for a number of reasons.

For starters, programming for the Gameboy was simply not worth it. It's interesting from a technical and perhaps a nostalgic perspective, but the reality is that I was making a game for a dead console that only a few enthusiasts would ever see.

Programming at such a low level for such old hardware is not easy. Additionally, resources on Gameboy programming weren't exactly abundant in 2011. A lot of strange glitches cropped up during development, and can be seen in some of the earlier snapshots of the game. In short, making anything at all happen was slow and frustrating.

Aside from that, I was running out of clock cycles. C was too bulky for the Gameboy Color. It dawned on me that if I were to make a proper game out of this, I would need to redo everything in assembly. That was the breaking point.

A word of advice

If you are a would-be gameboy game programmer, here is my advice to you: don't do it. It isn't worth the trouble for something that so few people will be able to see.

Lately I've been using Javascript and HTML5 to make games that can be played right inside a browser! Do that! Javascript is great!
